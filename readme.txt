The following folder provides dataset and code for running SI and baseline samplers. 

----- nsf-sampling.ipynb ---- 

An example runner for loading and running different samplers. The sampled nodes are saved to a file in directory /nsf_out/ 

----- nsf_data -----

This folder provides pickled version of nsf graphs by different organization. 

------ algos ------

This directory contains the sampling code for SI and baseline samplers.
